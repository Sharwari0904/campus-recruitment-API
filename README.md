# Campus Recruitment System

## Overview
The Campus Recruitment System is a full-stack web application designed to simplify the recruitment process for colleges.  
It provides separate dashboards for **Students** and **Admins**, allowing efficient job management, application tracking, and interview scheduling.

---

## Features

### Student
- View available job postings
- Apply for jobs
- Track application status
- View scheduled interviews

### Admin
- Add and manage job postings
- View registered students
- View and delete student applications
- Schedule interviews
- Monitor overall recruitment activity

## Functionalities Implemented
- Role-based access control (Student / Admin)
- CRUD operations for jobs and applications
- Secure data storage and retrieval
